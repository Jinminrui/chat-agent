## ADDED Requirements

### Requirement: 仓库校验脚本必须真实可执行
系统 MUST 提供真实可执行的仓库级校验命令，至少覆盖当前范围内声明支持的测试与静态检查入口。

#### Scenario: 在仓库根目录运行校验命令
- **WHEN** 开发者在仓库根目录执行项目声明的校验脚本
- **THEN** 脚本运行真实检查流程，而不是仅输出占位文本

### Requirement: 项目文档必须与实际启动和验证方式一致
系统 MUST 确保 README 中记录的启动、测试、环境变量与脚本命令与当前仓库实现保持一致。

#### Scenario: 新开发者按 README 操作
- **WHEN** 开发者仅根据 README 中的说明启动项目或运行验证命令
- **THEN** 文档中的命令、路径和前提条件与仓库中的真实配置一致
